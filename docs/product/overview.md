# Product Overview

## Introduction
AI Contests Navigator is an automated aggregation platform for AI competition information. It regularly scrapes major AI contest websites, uses AI technology to summarize and categorize contest information, and generates responsive static websites to publish this information.

## Core Value
- **Centralized Info**: One place for all AI contests.
- **AI-Enhanced**: Summaries and tagging powered by AI.
- **Developer-Focused**: API access and CLI tools.
- **High Performance**: Static site generation for speed.

## User Segments
- **AI Practitioners**: Searching for new challenges.
- **Students**: Looking for learning opportunities.
- **Researchers**: benchmarking models.

## Feature Highlights
1.  **Multi-Source Aggregation**: ModelScope, Civitai, Kaggle (planned), etc.
2.  **Smart Filtering**: By status, type (CV, NLP, Multi-modal), and difficulty.
3.  **Clean UI**: Modern, dark-themed (Catppuccin) interface.
4.  **Open Data**: Accessible via API and JSON.

## Usage
Users can visit the website to browse contests, or use the provided API for integration. Developers can run the scraper locally to get fresh data.
